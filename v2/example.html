AdTec v2 — GPT Ad Manager com SRA

Demonstração de todos os tipos de banner IAB via data-tam

Leaderboard (728x90)

Uso: <div class="adtec" data-tam="leaderboard"></div>

leaderboard — 728x90

Billboard (970x250) com targeting de channel

Uso: <div class="adtec" data-tam="billboard" data-channel="esportes"></div>

billboard — 970x250 — channel=esportes

Super Leaderboard (970x90)

Uso: <div class="adtec" data-tam="super-leaderboard"></div>

super-leaderboard — 970x90

Banner Clássico (468x60)

Uso: <div class="adtec" data-tam="banner"></div>

banner — 468x60

Multi-size: Leaderboard + Billboard no mesmo slot

Uso: <div class="adtec" data-tam="leaderboard,billboard"></div>

multi-size — 728x90 ou 970x250

Tamanho custom via WxH

Uso: <div class="adtec" data-tam="300x100"></div>

custom — 300x100

Targeting customizado com data-target-*

Uso: <div class="adtec" data-tam="medium-rectangle" data-author="joao" data-target-secao="politica"></div>

medium-rectangle — author=joao, secao=politica

Medium Rectangle (300x250)

data-tam="medium-rectangle"

medium-rectangle — 300x250

Half Page (300x600)

data-tam="half-page"

half-page — 300x600

Wide Skyscraper (160x600)

data-tam="wide-skyscraper"

wide-skyscraper — 160x600

Large Rectangle (336x280)

data-tam="large-rectangle"

large-rectangle — 336x280

Mobile Banner (320x50)

Uso: <div class="adtec" data-tam="mobile-banner"></div>

mobile-banner — 320x50

Mobile Banner Large (320x100)

Uso: <div class="adtec" data-tam="mobile-banner-large"></div>

mobile-banner-large — 320x100

Ad Unit customizado por slot (data-unit)

Uso: <div class="adtec" data-tam="medium-rectangle" data-unit="/rede/site/especial"></div>

medium-rectangle — ad unit custom: /rede/site/especial

Como Usar

<!-- 1. Adicione as divs com classe "adtec" e data-tam -->
<div class="adtec" data-tam="leaderboard"></div>
<div class="adtec" data-tam="medium-rectangle" data-channel="esportes"></div>
<div class="adtec" data-tam="300x600"></div>

<!-- 2. Inclua o script e inicialize -->
<script src="adtec.min.js"></script>
<script>
  AdTec.init({
    networkCode: '12345678',     // Código da rede (obrigatório)
    adUnitPath:  '/rede/site',    // Caminho da ad unit (obrigatório)
    collapseEmpty: true,           // Ocultar divs sem anúncio
    targeting: {                    // Targeting global (opcional)
      channel: 'esportes',
      author:  'joao'
    }
  });
</script>

API JavaScript

// Inicializar
AdTec.init({ networkCode: '12345678', adUnitPath: '/rede/site' });

// Refresh de todos os slots
AdTec.refresh();

// Refresh de slots específicos
AdTec.refresh(['adtec-slot-0', 'adtec-slot-2']);

// Ver slots definidos (debug)
console.table(AdTec.getSlots());

// Versão
console.log(AdTec.version); // "2.0.0"
